防火墙拦截、公司网连不上、能看不能控、卡顿延迟高、免费版 30 帧等症状，查 L1 官方排查口径 +
L2 网络层深度真因（P2P vs 转发、NAT 穿透、端口段、帧率档），对照真实场景快速恢复连接。无需安装，开网页即用。

**在线使用 → https://sunlogim.github.io/**

企业网端口段综合自公开远控社区经验（非官方）。更多见 [向日葵连接失败排查教程](https://sunlogim.com/lianjie/)。
